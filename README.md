# ixion-cli
Pentesting (penetration testing) AI assistant app

## setup
### clone the repo
```git clone https://github.com/guy-977/ixion-cli```\
```cd ixion-cli```
### install requirements
```pip install -r requirements.txt```\
```echo "OPENAI_API_KEY=YOUR ACTUAL API KEY" > .env```
### get started
```python main.py```
